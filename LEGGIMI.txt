CASA APP - V1.4 FLAT

Questa versione è senza sottocartelle.
Carica tutti i file nella root della repository GitHub, allo stesso livello di index.html.

IMPORTANTE AGGIORNAMENTO DA VERSIONI PRECEDENTI:
Se su GitHub Pages continui a vedere la vecchia versione, apri l'app una volta aggiungendo ?v=1.4 alla fine dell'indirizzo.
Esempio:
https://tuo-utente.github.io/nome-repository/?v=1.4

Perché:
le versioni precedenti potevano restare salvate nella cache del browser o nel service worker.
Questa versione disattiva i vecchi service worker e svuota le cache dell'app.

Novità V1.4:
- avvio vuoto senza dati demo;
- schermata iniziale CONFIGURA CASA se non esistono piani o stanze;
- dopo Reset dati ritorna alla configurazione iniziale;
- eliminata la funzione Ripristina dati demo;
- gestione piani solo da OPZIONI o dal primo setup;
- gestione stanze solo da OPZIONI o dal primo setup;
- modifica/elimina/aggiungi piano;
- modifica/elimina/aggiungi stanza;
- nuova chiave dati locale per evitare che dati demo vecchi mascherino la nuova versione;
- cache busting sui file CSS/JS per GitHub Pages.

Uso:
1. Apri index.html oppure pubblica su GitHub Pages.
2. Crea almeno un piano.
3. Crea almeno una stanza associata al piano.
4. Premi Entra nell'app.
5. Da OPZIONI puoi gestire piani, stanze, JSON e tema.
